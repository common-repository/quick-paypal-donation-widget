Thanks for installing  Paypal Donation Widget - Easy and simple way to display donation button on wordpress website.

== Description ==

Demo: http://demo.junktheme.com/wpdemo/paypal-donation/

Looking for a Donation Button for your website. PayPal donation is the best solution. Now you can setup donation button much easier way using our PayPal donation widget.

Features:
Easy to install and use.
Donate person can input custom amount if they wish. Also you can put a fixed amount.
All PayPal accepted currencies are available to select from.
Can select custom image donate button.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quick-paypal-donation-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to ”Appearance -> Widget”. Active the paypal Widget and Configure the setting.
